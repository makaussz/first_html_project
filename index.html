The Basic Language of the Web: HTML

Posted by Laura Jones on Monday, February 2030

All modern websites and web applications are built using 3 fundamental technologies: HTML, CSS, Javascript.These are languages of the web. In this post we are going to focus on HTML

What is HTML?

Hypertext Markup Language is a language web developers use to describe and stucture a web page.

HTML describes different types of content; Paragraphs, Links, Headings, Images. Web browsers understand HTML and renders HTML code as websites.

In HTML, each element is made up of 3 parts

  1. Opening Tag
  2. Closing Tag
  3. Actual content

You can learn more at MDN Web Docs

Why should you learn HTML?

There are countless reasons for learning HTML. Here are some of them;

  • To be able to use the fundamental web dev language
  • To handcraft beautiful websites
  • To build web applications
  • Impress friends
  • Have fun
